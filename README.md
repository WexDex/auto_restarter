# ⏻ Auto Restart Utility

A simple desktop tool that **automatically restarts your PC and selected applications at configurable intervals**.

---

## ✨ Features

* 🖥 **PC Auto Restart**
  Schedule a daily system restart with a live countdown.

* ⚙ **App / Backend Restart**
  Restart a chosen process every **X hours**. Supports `.exe` and `.bat` files.

* 🔁 **Manual Restart**
  Instantly restart the selected application.

* 🟢 **Status Indicator**
  Shows **ACTIVE / STOPPED** state for backend auto-restarts.

* 📝 **Logging**
  All restarts are recorded in:

```text
restart_log.txt
```

---

## 🚀 Run

```bash
python auto_restart.py
```

---

## 📁 Files

```text
auto_restart.py   Main application
config.json       Saved settings
restart_log.txt   Restart history
```
